# ☕ Coffee Sales Data Analysis (Excel)

Turn raw coffee sales data into meaningful insights using Excel! This project showcases data cleaning, analysis, and interactive dashboards to understand sales trends and business performance.

* Data cleaning & preprocessing
* Pivot tables & charts
* Interactive dashboard
* Sales trends & top products

## 🚀 Insights

* Identified best-selling coffee products
* Analyzed peak sales periods
* Tracked revenue patterns

## 🛠 Tools Used

* Microsoft Excel
